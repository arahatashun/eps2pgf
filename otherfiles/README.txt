Introduction
----------------------------------------------------------------------------
Convert Encapsulated PostScript (EPS) graphics files to the Portable
Graphics Format (PGF, http://sourceforge.net/projects/pgf/) for inclusion in
LaTeX documents. Texts in the figure are typeset by LaTeX
